# Koh-lipe-redesign
A responsive multi-page web application for Koh Lipe, an award-winning Southern Thai restaurant. Features include an interactive booking calendar, dynamic menu sections for multiple locations, and a modern, mobile-friendly design.

# Project Title: Koh Lipe Thai Cuisine Website
# Key Features:

Interactive Reservation System: Includes a custom-built JavaScript calendar for date selection and reservation details (location, time, and guest count).
Dynamic Menu Displays: Segregated menu sections for Special, Regular, Keto, Drinks, and Desserts across different restaurant locations (Toronto and Scarborough).
Responsive Design: Fully styled with CSS Grid and Flexbox to ensure a seamless experience across desktop and mobile devices.
Michelin Guide Integration: Showcases the restaurant's Michelin recognition from 2022–2025.

# Technologies Used:

HTML5: Semantic structure for restaurant information and menus.
CSS3: Custom styling using Google Fonts (Inter, Dancing Script, Kaisei HarunoUmi) and FontAwesome icons.
JavaScript (ES6): Logic for the calendar functionality, navigation, and menu section switching.
